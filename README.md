# 8 Week SQL Challenge – Analytics Case Studies

This repository contains my solutions to the **8 Week SQL Challenge** by Danny Ma.

Rather than focusing only on correct outputs, I approach each case study from a
**real-world analytics perspective** — considering edge cases, business rules,
query design trade-offs, and reusability.

## What this repo includes
- Table schemas and sample data
- Core and bonus SQL solutions
- Reusable analytical views
- Notes on assumptions and edge cases
